# World Explorer

World Explorer is a Next.js project that allows users to explore countries around the world.
<img width="1861" height="898" alt="image" src="https://github.com/user-attachments/assets/9f63edfb-a9b6-4fcf-af67-a3d3f5230fb9" />

## Features

- App Router
- File-based routing
- Shared layout
- Dynamic routes
- Server components
- Client components
- Real API data fetching
- Static rendering and caching
- Dynamic rendering
- Search functionality

## API Used

REST Countries API  
https://restcountries.com/v2/all?fields=name,flags,region,population,alpha3Code
